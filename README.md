dice-service
============

Dice rolling as a service. This is a simple web service to roll 
dice and return a result set.

It's mostly intended to be a demonstration of building a 
deployable application for [Docker](http://www.docker.io)

Requirements
------------

    bottle==0.11.6
    diceroll==2.4
    gunicorn==17.5
    pyparsing==2.0.1
